Projects

Migrate QlikView to QlikSense

During this project, my role involved supporting the Data Architect/BI developer in collaboration with Business Analysts to facilitate the transition from QlikView to Qlik Sense. Our primary objective was to migrate all existing applications from the old version to the new one while also creating new applications, visualizations, and reports. Additionally, I conducted training sessions for users from various departments within the organization to ensure smooth adoption of the new tool. As part of my responsibilities, I provided initial support for the tool and developed reports based on the specific needs of the business.

ManageEngine Opmanager

The organization made significant investments in the ManageEngine stack, which offers a range of products including service desk and Desktop Central for remote control and device management. The project's goal was to update all assets in the network monitoring tool (Opmanager) to effectively visualize our network. Additionally, it aimed to assist the infrastructure team in proactively addressing any issues before users/clients raise tickets to the service desk.

Projects

Cyber Security - Wi-Fi Hacking with a Raspberry Pi

The objective of this project is to construct a hacking tool using a raspberry pi in order to raise awareness about Cyber Security and showcase the ease of connecting to or compromising a Wi-Fi network with inadequate encryption. The project will be considered successful if I am able to establish a connection to a Wi-Fi network with weak encryption, gather details about the network, and obtain information about the devices connected to it.

Capstone Project - Parking web

This project emerged as a remedy for students facing difficulties in locating vacant parking spaces, leading us to devise a parking website as a solution. It was a collaborative effort involving a group, where my role was that of a web developer. The project was implemented utilizing CSS, HTML, PHP, and XAMPP.
